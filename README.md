# Plus1

![image](https://img.itch.zone/aW1nLzE3MDQwNjIucG5n/original/0nWxzc.png)

## Simple 2D platformer cooperative game made in Unity

### CONTROLS

#### Player 1

Movement - Left/Right Arrow Keys

Look Up/Down - Up/Down Arrow Keys

Jump - J

Shoot - K

#### Player 2

Movement - A/D

Look Up/Down - W/S 

Jump - C

Shoot - V
